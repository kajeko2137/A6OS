The Raspberry Pi 1 B uses the BCM2835 SoC.
The base address for peripherals is 0x20000000.
The UART0 (PL011) base address is 0x20201000.
We need to configure GPIO pins 14 (TXD) and 15 (RXD) for alternate function 0 to use UART0.
The GPIO base address is 0x20200000.

Registers to use:
- GPIO_FSEL1 (0x20200004): Set pins 14 and 15 to ALT0 (0b100)
- GPIO_PUD (0x20200094) and GPIO_PUDCLK0 (0x20200098): Disable pull-up/pull-down for pins 14 and 15.
- UART0_CR (0x20201030): Disable UART before configuration, then enable.
- UART0_IBRD (0x20201024) and UART0_FBRD (0x20201028): Set baud rate (115200 typically, given a UART clock of 3MHz, IBRD=1, FBRD=40).
- UART0_LCRH (0x2020102c): Set 8-bit word length, enable FIFOs.
- UART0_DR (0x20201000): Write data here to transmit.
- UART0_FR (0x20201018): Check TXFF (Transmit FIFO Full) flag (bit 5) before writing.

Steps:
1. Initialize UART0 in a `uart_init` function.
2. Create a `uart_send` function that waits for !TXFF and writes a byte to UART0_DR.
3. Create a `uart_puts` function that loops through a null-terminated string and calls `uart_send`.
4. In `_start`, call `uart_init`, then load the address of your string and call `uart_puts`.
